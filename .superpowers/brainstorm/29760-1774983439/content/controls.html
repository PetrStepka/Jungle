Ovládání

Jak jednoduché nebo komplexní má být ovládání?

A

Jednoduché (4 klávesy)

Šipky = pohyb doleva/doprava + skok. Z = melee útok, X = střelba. To je vše. Rychle se naučíš, hned hraješ.

B

Střední (6 kláves)

WASD = pohyb + skok. J = melee, K = střelba, L = speciální schopnost (dash / úder). Víc možností, trochu strmější křivka učení.

C

Klávesnice + myš

WASD = pohyb. Myš = míření a střelba (levé tlačítko), melee (pravé tlačítko). Přesnější míření, modernější feel.
